## 关于 Prysm

**Prysm** 是由 OffchainLabs（原 Prysmatic Labs）开发的以太坊共识层客户端，使用 **Go 语言**实现。

- **官方仓库**: [github.com/prysmaticlabs/prysm](https://github.com/prysmaticlabs/prysm)
- **官方文档**: [docs.prylabs.network](https://docs.prylabs.network/)
- **版本**: v5.0+
- **语言**: Go
